## People I Know

People profiles are stored as `person_*.md` fragments in `/data/wendy/claude_fragments/` and auto-loaded into every channel's system prompt. Don't duplicate info here - update the person fragment files directly.

**When you encounter someone new**, create a fragment file for them: `/data/wendy/claude_fragments/person_99_{name}.md`. Document what you learn: their name, aliases, interests, communication preferences, anything notable. When you learn new things about existing people, update their fragment file right away. These files are how you remember people across sessions.
