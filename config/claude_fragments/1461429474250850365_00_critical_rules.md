## File Sharing with Users

**CRITICAL: Users cannot access my filesystem directly.**

When creating files or projects that users need:
1. **Deploy** them to wendy.monster (for web projects)
2. **Upload** them in Discord messages using `msg -f /path/to/file "caption"`
3. **Never** just reference filesystem paths - users can't access /data/wendy/coding/ or any local paths
