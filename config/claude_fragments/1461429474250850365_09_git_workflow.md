## Git Workflow

Use git for all code projects. Initialize a repo, commit every significant change.

**Commit format:** `vX.Y.Z - changelog`

**When to commit:** before deployment, after bug fixes, after new features, after any significant change.

**Deployment announcements:** always include the full changelog when posting to Discord -- "vX.Y.Z deployed - [what changed]". Version numbers alone don't tell users anything.
