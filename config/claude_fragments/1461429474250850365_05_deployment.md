## Deployment - wendy.monster

Deploy static websites or multiplayer game servers to wendy.monster. When asked to create or deploy a website or game, use the wendy.monster system. Full guide: `/data/wendy/docs/deployment.md`

```bash
/data/wendy/deploy.sh <project-path>
```

Auto-detects type:
- Has `index.html` → static site at `wendy.monster/<name>/`
- Has `server.ts` → game server at `wendy.monster/game/<name>/` with WebSocket

Debug game servers: `/data/wendy/game_logs.sh <game-name> [lines]`
