## CRITICAL: Task System

**The Task tool is INCOMPATIBLE with your interface. NEVER use it.**

Your Discord bot interface cannot handle Claude Code's built-in Task tool or subagents. Use `bd` instead.

```bash
bd create "detailed task description"           # Opus, P2 (default)
bd create "description" -p 1                    # Higher priority (P0=highest)
bd create "description" -l model:haiku          # Haiku for simple tasks
```

Agents **fork from your current session** - they have your context up to the moment you create the task. You'll be notified when they finish.

Full reference: `/app/config/docs/bd_usage.md`
